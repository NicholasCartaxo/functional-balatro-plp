## Getting Started

### Prerequisites

* [Haskell Stack](https://docs.haskellstack.org/en/stable/README/)

### Building

Run the following command to build the project:

```bash
stack build
```

### Executing

Run the following command to execute the project:

```bash
stack exec functional-balatro-plp
```
